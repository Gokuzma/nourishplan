  <files>src/hooks/useHousehold.ts, src/pages/HouseholdSetup.tsx, src/components/household/CreateHousehold.tsx, src/components/household/JoinHousehold.tsx</files>
  <action>
1. **useHousehold hooks** (`src/hooks/useHousehold.ts`):
   Create TanStack Query hooks following Pattern 5 from 01-RESEARCH.md:

   - `useHousehold()` — `useQuery` that fetches the current user's household via `household_members` join `households`. Returns null if user has no household.
   - `useHouseholdMembers()` — `useQuery` that fetches all members of the user's household, joined with `profiles` for display names.
   - `useCreateHousehold()` — `useMutation` that:
     a. Inserts into `households` (name)
     b. Inserts into `household_members` (household_id, user_id, role: 'admin')
     c. Invalidates `['household']` query cache on success
   - `useJoinHousehold(token)` — `useMutation` that:
     a. Reads `household_invites` where `token` matches, `expires_at > now()`, `used_at is null`
     b. If valid: inserts into `household_members` (household_id from invite, user_id, role: 'member')
     c. Updates `household_invites` to set `used_at = now()`
     d. Invalidates `['household']` cache
   - `useCreateInvite()` — `useMutation` that inserts into `household_invites` (household_id, created_by), returns the token.

2. **HouseholdSetup page** (`src/pages/HouseholdSetup.tsx`):
   - Per locked decision: "Post-signup screen presents Create or Join fork".
   - Two-card layout: "Create a Household" card and "Join a Household" card.
   - If URL has `?invite=<token>` query param, auto-select the Join flow and pre-fill token.
   - After successful create or join, navigate to home page `/`.
   - Style with pastel theme: cream background, sage green buttons, peach accent borders.

3. **CreateHousehold component** (`src/components/household/CreateHousehold.tsx`):
   - Simple form: household name input + "Create" button.
   - Calls `useCreateHousehold` mutation.
   - Shows loading state during creation.
   - Error handling for failed insert.

4. **JoinHousehold component** (`src/components/household/JoinHousehold.tsx`):
   - If token from URL: show "Join {household_name}?" confirmation.
   - If no token: show input field to paste invite token/link.
   - Calls `useJoinHousehold` mutation.
   - Error states: expired token, already used, already in a household.
