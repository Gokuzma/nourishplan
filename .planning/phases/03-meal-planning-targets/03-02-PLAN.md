  <name>Task 1: Meals CRUD hooks</name>
  <files>src/hooks/useMeals.ts</files>
  <action>
    Create `src/hooks/useMeals.ts` following the exact pattern of `src/hooks/useRecipes.ts`:

    **useMeals():** Query all meals for household (where deleted_at is null), ordered by name. Query key: `['meals', householdId]`. Enabled when householdId exists.

    **useMeal(id):** Query single meal with its meal_items joined. Use `.select('*, meal_items(*)')`. Query key: `['meal', id]`. Enabled when id is truthy.

    **useCreateMeal():** Insert into `meals` table with household_id from `useHousehold()` and created_by from `useAuth()`. Invalidate `['meals', householdId]` on success. Return the created meal so caller can navigate to `/meals/{id}`.

    **useUpdateMeal():** Update meal name. Invalidate both `['meals', householdId]` and `['meal', id]`.

    **useDeleteMeal():** Soft-delete via `update({ deleted_at: new Date().toISOString() })`. Invalidate `['meals', householdId]`.

    **useAddMealItem():** Insert into `meal_items` with meal_id, item_type, item_id, quantity_grams, and per-100g macro snapshots (calories_per_100g, protein_per_100g, fat_per_100g, carbs_per_100g — extracted from the NormalizedFoodResult at add-time). Set sort_order to `items.length`. Invalidate `['meal', mealId]`.

    **useUpdateMealItem():** Update quantity_grams on a meal_item. Invalidate `['meal', mealId]`.

    **useRemoveMealItem():** Delete meal_item row. Invalidate `['meal', mealId]`.

    All mutations use `useMutation` from TanStack Query. All queries use `useQuery`.
