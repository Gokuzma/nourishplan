  <action>
    Create `src/hooks/useNutritionTargets.ts` following TanStack Query patterns from useRecipes.ts:

    **useNutritionTargets(householdId):** Query all nutrition_targets for the household. Query key: `['nutrition-targets', householdId]`. Returns array of NutritionTarget. Enabled when householdId truthy.

    **useNutritionTarget(householdId, memberId, memberType):** Query single member's target. `memberType` is 'user' | 'profile'. Query: `.eq('household_id', householdId)` and `.eq(memberType === 'user' ? 'user_id' : 'member_profile_id', memberId)`. Use `.maybeSingle()` to return null if no targets set yet. Query key: `['nutrition-target', householdId, memberId]`.

    **useUpsertNutritionTargets():** Mutation that upserts a nutrition_targets row. Accept `{ householdId, userId?, memberProfileId?, calories?, protein_g?, carbs_g?, fat_g?, micronutrients?, custom_goals? }`. Use Supabase `.upsert()` with `onConflict` matching the appropriate unique constraint (household_id,user_id for auth users or household_id,member_profile_id for managed profiles). Invalidate `['nutrition-targets', householdId]` and `['nutrition-target', householdId, memberId]` on success.

    All hooks import supabase from `../lib/supabase`, useAuth from `../contexts/AuthContext`, types from `../types/database`.
