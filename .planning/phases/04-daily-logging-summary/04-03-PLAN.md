   ```typescript
   workbox: {
     runtimeCaching: [
       {
         urlPattern: ({ request }) => request.mode === 'navigate',
         handler: 'NetworkFirst',
         options: {
           cacheName: 'html-cache',
           networkTimeoutSeconds: 3,
           expiration: { maxEntries: 5, maxAgeSeconds: 86400 },
           cacheableResponse: { statuses: [200] },
         },
       },
     ],
   },
   ```
   This enables app-shell caching so the app loads from cache when offline. Pre-caching of JS/CSS/assets is handled automatically by vite-plugin-pwa.

   - Prompt appears after a brief delay (e.g., 3 seconds after mount) to avoid interrupting first load — per user decision "shown after first action, non-intrusive"
   - Dismiss button that sets localStorage flag and hides
   - If already installed (standalone mode), never shows
   - Tailwind styled: subtle bottom bar or toast, not blocking
