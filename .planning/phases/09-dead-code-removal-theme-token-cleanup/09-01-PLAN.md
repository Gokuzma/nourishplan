Replace hardcoded amber Tailwind classes with project semantic theme tokens in `src/components/log/OfflineBanner.tsx`.

```
bg-amber-100 text-amber-800 text-sm px-4 py-2 text-center w-full
```
to:
```
bg-accent/20 text-text border-b border-accent/40 text-sm px-4 py-2 text-center w-full
```

Rationale per research: `--color-accent` is peach (#E8B4A2 light, #F0C4B2 dark) — already used throughout the app for subtle highlight backgrounds. `bg-accent/20` gives a 20% opacity tint that adapts automatically via the `.dark` class override in global.css. `text-text` adapts between #3D3D3D (light) and #E8E5E0 (dark). The `border-b border-accent/40` adds a subtle bottom border for visual separation.

Do NOT add a new `--color-warning` token — the accent color at reduced opacity is the established pattern for status indicators in this project.
