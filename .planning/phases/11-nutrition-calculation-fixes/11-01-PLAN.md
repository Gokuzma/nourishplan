**1. Fix FreeformLogModal.tsx ‚Äî change logging pattern to use real per-unit macros:**

In `handleLog()`, change from:
```typescript
servings_logged: 1,
calories_per_serving: (grams / 100) * selectedFood.calories,
```
To:
```typescript
servings_logged: quantity,
calories_per_serving: (selectedUnit.grams / 100) * selectedFood.calories,
protein_per_serving: (selectedUnit.grams / 100) * selectedFood.protein,
fat_per_serving: (selectedUnit.grams / 100) * selectedFood.fat,
carbs_per_serving: (selectedUnit.grams / 100) * selectedFood.carbs,
```

This means `calories_per_serving` holds the macros for ONE unit of the selected portion (e.g., 1 cup, 1 gram, 1 tbsp). `servings_logged` holds how many of that unit the user selected. `calcLogEntryNutrition(log)` = `per_serving * servings_logged` = correct total.

Also scale micronutrients by the unit grams:
```typescript
micronutrients: Object.fromEntries(
  Object.entries(selectedFood.micronutrients ?? {}).map(
    ([key, val]) => [key, (selectedUnit.grams / 100) * val]
  )
),
```

Also pass `serving_unit`:
```typescript
serving_unit: selectedUnit.description,
```
