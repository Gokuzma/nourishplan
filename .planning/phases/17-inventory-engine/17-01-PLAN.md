**Create src/utils/inventory.ts:**

Import `InventoryItem`, `InventoryUnit`, `StorageLocation` from `../types/database`.

1. `getExpiryUrgency(expiresAt: string | null): ExpiryUrgency`
   - Type: `'expired' | 'urgent' | 'warning' | 'ok' | 'none'`
   - If null, return 'none'
   - Use Date.UTC() for today and expiry (same pattern as getWeekStart in mealPlan.ts)
   - `const now = new Date(); const todayMs = Date.UTC(now.getUTCFullYear(), now.getUTCMonth(), now.getUTCDate())`
   - Parse expiresAt: `const [y,m,d] = expiresAt.split('-').map(Number); const expiryMs = Date.UTC(y, m-1, d)`
   - `daysUntil = Math.ceil((expiryMs - todayMs) / 86400000)`
   - < 0 = 'expired', <= 3 = 'urgent', <= 7 = 'warning', else 'ok'

2. `convertToGrams(quantity: number, unit: InventoryUnit): number | null`
   - g: quantity * 1, kg: quantity * 1000, ml: quantity * 1, L: quantity * 1000, units: return null

3. `computeFifoDeductions(items: InventoryItem[], needs: { food_id: string | null; food_name: string; quantity_grams: number }[]): { deductions: { item: InventoryItem; deductAmount: number }[]; missing: string[] }`
   - For each need:
     - Find matching active items (removed_at === null): first by food_id exact match (if food_id is non-null), then by food_name case-insensitive if no food_id match
     - Sort matches by purchased_at ASC (FIFO)
     - Deduct needed quantity across items: convert each item's quantity_remaining to grams via convertToGrams. If unit is 'units', skip (add to missing). Deduct from oldest first, tracking how much was deducted from each. If item runs out, move to next.
     - If no matches found at all, add food_name to missing array
   - Return { deductions, missing }

4. Phase 18 query interfaces (D-30):
   - `getAvailableQuantity(items: InventoryItem[], foodId: string): number` — sum of convertToGrams(quantity_remaining, unit) for items where food_id matches and removed_at is null. Skip 'units' items.
   - `getLowStockStaples(items: InventoryItem[], thresholdGrams = 100): InventoryItem[]` — filter where is_staple=true and convertToGrams(quantity_remaining, unit) is not null and < thresholdGrams
   - `getExpiringSoonItems(items: InventoryItem[], withinDays = 7): InventoryItem[]` — filter where getExpiryUrgency(expires_at) is 'urgent' or 'warning'
   - `getPurchaseHistory(items: InventoryItem[], foodId: string): { purchased_at: string; purchase_price: number | null; unit: InventoryUnit }[]` — filter by food_id (include removed items), return purchase info
