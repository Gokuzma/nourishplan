Create `src/hooks/useInventory.ts` with four hooks following the useFoodPrices pattern:

**1. useInventoryItems(location?: StorageLocation)**
- Get householdId from `useHousehold()` (same as useFoodPrices)
- Query key: `location ? queryKeys.inventory.byLocation(householdId, location) : queryKeys.inventory.list(householdId)`
- QueryFn: `supabase.from('inventory_items').select('*').eq('household_id', householdId!).is('removed_at', null).order('expires_at', { ascending: true, nullsFirst: false })`
- If location is provided, add `.eq('storage_location', location)` to the query
- Return type: `InventoryItem[]`
- enabled: `!!householdId`

**2. useAddInventoryItem()**
- Get session from `useAuth()`, membership from `useHousehold()`, queryClient
- Params type: `{ food_name: string; brand?: string; food_id?: string; quantity_remaining: number; unit: InventoryUnit; storage_location: StorageLocation; is_opened?: boolean; is_staple?: boolean; purchased_at?: string; expires_at?: string | null; purchase_price?: number | null; is_leftover?: boolean; leftover_from_recipe_id?: string | null }`
- MutationFn: insert into inventory_items with `household_id: householdId, added_by: userId, ...params`
- onSuccess: invalidate `['inventory', householdId]` (prefix invalidation clears all sub-keys)
- **D-04 price integration:** If `params.purchase_price` and `params.food_id` are both present AND `params.unit !== 'units'`, import `normaliseToCostPer100g` from `../utils/cost` and `useSaveFoodPrice` from `./useFoodPrices`. After the insert succeeds, call `saveFoodPrice.mutateAsync({ food_id: params.food_id, food_name: params.food_name, store: '', cost_per_100g: normaliseToCostPer100g(params.purchase_price, params.quantity_remaining, params.unit) })`.
  - NOTE: The saveFoodPrice call is a separate mutation inside onSuccess. Import useSaveFoodPrice at hook level.

**3. useUpdateInventoryItem()**
- Params: `{ id: string; updates: Partial<InventoryItem> }`
- MutationFn: `supabase.from('inventory_items').update(updates).eq('id', id).select().single()`
- onSuccess: invalidate `['inventory', householdId]`

**4. useRemoveInventoryItem()**
- Params: `{ id: string; reason: RemovalReason }`
- MutationFn: `supabase.from('inventory_items').update({ removed_at: new Date().toISOString(), removed_reason: reason }).eq('id', id)`
- onSuccess: invalidate `['inventory', householdId]`

All hooks import from: `@tanstack/react-query`, `../lib/supabase`, `./useAuth`, `./useHousehold`, `../lib/queryKeys`, `../types/database`.
