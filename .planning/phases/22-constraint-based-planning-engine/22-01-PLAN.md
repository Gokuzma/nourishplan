**Create src/utils/nutritionGaps.ts (GREEN phase):**

```typescript
export interface MemberIdentity {
  id: string
  type: 'user' | 'profile'
  name: string
}

export interface WeeklyGap {
  memberId: string
  memberName: string
  nutrient: string
  weeklyTarget: number
  weeklyActual: number
  percentOfTarget: number
}
```

`calcWeeklyGaps(slots, targets, members, threshold = 0.9)`:
1. For each member in `members`, find their `NutritionTarget` by matching `user_id` (type='user') or `member_profile_id` (type='profile')
2. If target is null or has no calories set, skip this member
3. Compute weekly totals from `slots`: iterate all slots, sum meal nutrition using the same per-100g math as `calcMealNutrition` — for each meal_item: `(quantity_grams / 100) * per_100g_value`
4. Weekly target = daily target * 7
5. For each macro (calories, protein, carbs, fat): if `weeklyActual / weeklyTarget < threshold`, add a `WeeklyGap` entry
6. Return the array of gaps

The function must NOT import supabase or any hooks — pure function only.
