**Function signature:**
```typescript
import type { SwapSuggestion } from '../components/plan/NutritionGapCard'
import type { WeeklyGap } from './nutritionGaps'
import type { SlotWithMeal } from '../hooks/useMealPlan'
import type { Meal, MealItem } from '../types/database'

export function computeSwapSuggestions(
  gaps: WeeklyGap[],
  slots: SlotWithMeal[],
  allMeals: (Meal & { meal_items: MealItem[] })[],
  weekStart: string,
  weekStartDay: number,
): SwapSuggestion[]
```

**Algorithm:**
1. For each gap (member + nutrient), find the best single-slot swap that maximises the nutrient gain:
   a. Iterate all unlocked slots that have a meal assigned (`slot.is_locked !== true && slot.meal_id != null`).
   b. For each such slot, compute its current contribution to the gap nutrient using `calcIngredientNutrition` + `calcMealNutrition` (same pattern as `calcSlotNutrition` in nutritionGaps.ts — duplicate the 4-line helper locally, do not import the private function).
   c. For each candidate meal from `allMeals` that is NOT already assigned to any slot in the plan, compute its nutrient value for the gap nutrient.
   d. The gain = candidateMealNutrient - currentSlotNutrient. Track the best (slot, candidateMeal) pair with highest gain.
2. If best gain > 0, build a `SwapSuggestion` with the slot's dayIndex, slotName, the candidate meal's name, the gain, and the gap's nutrient.
3. Derive `dayName` from `dayIndex` + `weekStart` + `weekStartDay`: compute the actual date from weekStart + dayIndex, then use `['Sun','Mon','Tue','Wed','Thu','Fri','Sat'][date.getUTCDay()]`.

**Important:** The `SwapSuggestion` interface is already defined and exported from `NutritionGapCard.tsx`. Import it from there — do NOT redefine it.

```typescript
const swapSuggestions = useMemo(() => {
  if (gaps.length === 0 || !slots.length || !meals.length) return []
  // allMeals need meal_items with full macros — slots.meals has them, but the meals hook only fetches calories.
  // Build full meal list from slots data (which has meal_items with all 4 macros).
  const fullMeals = slots
    .filter(s => s.meals)
    .map(s => s.meals!)
    .filter((m, i, arr) => arr.findIndex(x => x.id === m.id) === i)
  // Also include meals not currently in slots (from the meals list) — but meals hook only has calories_per_100g.
  // For swap suggestions, we can only reliably use meals that appear in slots (which have full macro data).
  // Plus all household meals that have meal_items with full macros.
  return computeSwapSuggestions(gaps, slots, fullMeals, weekStart, weekStartDay)
}, [gaps, slots, meals, weekStart, weekStartDay])
```
