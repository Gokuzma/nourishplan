**Current bug:** `src/components/plan/PlanGrid.tsx` lines 574-604 contain two nearly-identical render blocks (mobile carousel and desktop stack) that use the pattern:
```tsx
{DEFAULT_SLOTS.map(slotName => {
  const slot = displaySlots.find(s => s.day_index === i && s.slot_name === slotName)
  return slot?.is_locked
    ? dayCards[i]                     // <-- BUG: returns the ENTIRE day card (all 4 slots)
    : <SlotShimmer key={slotName} />

```
