**Input type detection function (per D-04):**
```typescript
function detectInputType(input: string): 'youtube' | 'url' | 'text' {
  const trimmed = input.trim();
  if (/^https?:\/\/(www\.)?(youtube\.com\/watch|youtu\.be\/)/.test(trimmed)) return 'youtube';
  if (/^https?:\/\//.test(trimmed)) return 'url';
  return 'text';
}
```

**Blog URL fetch (per D-05, Research Pattern 5):**
- Use `Deno.fetch(url)` with `AbortController` 10-second timeout
- Set `User-Agent: Mozilla/5.0 (compatible)` header
- Strip HTML: remove `<script>` and `<style>` blocks, then all tags, collapse whitespace
- Truncate to 12000 chars
- If stripped text < 200 chars, return error per D-10: "Could not fetch that URL. Try copying and pasting the recipe text directly instead."

**YouTube transcript fetch (per D-06, Research Pattern 4):**
- Fetch `https://www.youtube.com/watch?v=VIDEO_ID` with `Accept-Language: en` header and 10s abort
- Extract `ytInitialPlayerResponse` JSON from page HTML via regex: `/var ytInitialPlayerResponse\s*=\s*(\{.*?\});/s`
- Parse JSON, navigate to `captions.playerCaptionsTracklistRenderer.captionTracks[0].baseUrl`
- Fetch that URL to get XML timed-text
- Extract text content from `<text>` elements: `html.match(/<text[^>]*>(.*?)<\/text>/gs)` and strip tags
- Join all text segments with spaces, truncate to 12000 chars
- If any step fails, return D-10 error

**Raw text:** Use input directly (no fetch), truncate to 12000 chars.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client -> Edge Function | Untrusted user input (URL or text) crosses into server |
| Edge Function -> External URLs | Server fetches arbitrary user-supplied URLs |
| Edge Function -> Anthropic API | User-sourced content sent to AI model |
| Edge Function -> Supabase DB | AI-generated data written to database |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-25-01 | Tampering | AI prompt | mitigate | User input goes into AI `user` message content only; system prompt is a server-side constant never interpolated with user data |
| T-25-02 | Elevation of Privilege | SSRF via URL fetch | accept | Deno edge functions run in sandboxed network context; Supabase blocks private subnet access; AbortController limits fetch duration to 10s |
| T-25-03 | Tampering | Unauthorized recipe creation | mitigate | Identity extracted from JWT via adminClient.auth.getUser(token); never from request body (L-013); household membership verified before any INSERT |
| T-25-04 | Information Disclosure | Raw HTML in DB | mitigate | Only plain text and numbers stored; no HTML markup reaches DB columns; AI output parsed as JSON with specific typed fields |
| T-25-05 | Denial of Service | Large URL response | mitigate | HTML/transcript truncated to 12000 chars; AbortController 10s timeout on all external fetches; max_tokens 2048 on AI call |
</threat_model>
