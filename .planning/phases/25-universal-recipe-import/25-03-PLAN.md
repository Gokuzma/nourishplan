<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 2: Verify complete import flow</name>
  <what-built>
Universal recipe import feature: Import Recipe button on RecipesPage, modal with textarea input, AI-powered extraction from blog URLs/YouTube URLs/raw text, auto-save to RecipeBuilder with ingredients, macros, cooking steps, and source URL attribution.
  </what-built>
  <how-to-verify>
**Test 1: Blog URL import**
1. Go to the Recipes page
2. Tap "Import Recipe" (outlined button next to "+ New Recipe")
3. Paste a blog recipe URL (e.g., `https://www.allrecipes.com/recipe/24002/easy-homemade-chicken-pot-pie/` or any recipe blog post)
4. Tap "Import Recipe" button
5. Confirm: spinner shows "Importing…" while processing
6. Confirm: after completion, you are navigated to RecipeBuilder
7. Confirm: recipe has a name, ingredients with quantities and macros, and cooking steps
8. Confirm: "Imported from [domain]" attribution link appears below recipe name
9. Confirm: clicking the attribution link opens the original URL in a new tab

**Test 2: YouTube URL import**
1. Go back to Recipes page
2. Tap "Import Recipe"
3. Paste a YouTube cooking video URL (e.g., a popular cooking channel video)
4. Tap "Import Recipe" button
5. Confirm: recipe extracts with ingredients and steps from the video transcript
6. Note: If YouTube transcript extraction fails, the modal should show an error suggesting to paste text instead. This is expected behavior (YouTube format may change).

**Test 3: Raw text import**
1. Go back to Recipes page
2. Tap "Import Recipe"
3. Paste raw recipe text, e.g.:
```
Spaghetti Carbonara
Serves 4
Ingredients:
400g spaghetti
200g guanciale, diced
4 egg yolks
100g pecorino romano, grated
Black pepper to taste
Instructions:
1. Cook spaghetti in salted boiling water until al dente
2. Fry guanciale until crispy
3. Mix egg yolks with pecorino
4. Toss hot pasta with guanciale, then egg mixture off heat
5. Season with black pepper and serve
```
4. Tap "Import Recipe"
5. Confirm: recipe appears in RecipeBuilder with all ingredients and steps populated

**Test 4: Error handling**
1. Tap "Import Recipe"
2. Paste an invalid URL like `https://thisdomaindoesnotexist12345.com/recipe`
3. Confirm: error message appears in modal (not a crash)
4. Confirm: you can edit the input and retry or close the modal

**Test 5: Source URL visibility**
1. From any imported recipe in RecipeBuilder, confirm "Imported from [domain]" link is visible
2. Go to the Recipes list page and confirm NO source badge appears on recipe cards (D-13)
  </how-to-verify>
